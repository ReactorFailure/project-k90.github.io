What is HTML and CSS?

HTML stands for Hypertext Markup Language and CSS stands for Cascading Stylesheet. HTML is used for constructing the skeleton of the website while its partner, CSS, helps style the website. They are both different languages but we can connect them together which I will be showing you soon. I am using the latest versions (hopefully) of HTML 5 and CSS 3.

Getting Started

To get started, you will need a coding editor or an IDE (Integrated Develpement Environment, basically a coding editor). The IDE I'm using is Visual Studio Code, however there many other programming editors out there you can use such as Notepad ++, Atom, etc. It's all about preference. Another thing you will need is to find the TAB key on your keyboard which should be above the Caps Lock and it should have a symbol with 2 arrows pointing at opposite directions. The TAB key will help you create 4 spaces without you having to press the space bar 4 times. This is called "indenting" and many programmers use it to make their code look nicer. However another reason why we use indenting is because ome programming languages such as Python which some of its functions require an indent or else it will give you an error. If you press the spacebar 4 times to indent, you are a threat to society.

Making your HTML and CSS files

  1. Make a folder on your desktop and name it "My first website" or something.
  2. Inside that folder, make another folder and name it "CSS". If you haven't guessed already, this is where you will put your CSS script inside.
  3. Right-click you main website folder (not the CSS folder) and click "Open with [IDE of your choice]".
  4. Once you open your IDE, look at your file explorer (which is usually on the left, at least for me), find and click on your CSS folder and add a file there called "MyFirstCSS.css" (or name it whatever you like but keep the ".css" at the end).
  5. Make another file called "MyFirstHTML.html" (name it whatever you like but keep the ".html" at the end) and move it outside your CSS folder.

You have now created a basic file path for your first website

Your fist HTML script

Firstly, you will have to open your html file with your IDE. Once you open the file type out this code (DON'T FORGET TO INDENT!):

                <!DOCTYPE html>
                <html>
                    <head></head>
                    <body></body>
                </html>

Lets do some analysis, <!DOCTYPE html> tells the computer that we are using HTML and not some other language. The <head> tag is a tag where you put your website's data such as "metadata", "webite title", "stylesheet", etc. The <body> tag is where you put your website's content such as headings, paragraphs, etc. As you may have noticed, there are some duplicate tags but have a "/" in them. These are end tags. They are basically the "period" for HTML. It tells when the tag should stop. However there are some tags that do not need an end tag such as the doctype.

Adding tags to the Head Tag

Inside the head tag, we can add tags to link to our CSS file and make a title to our website like this:

                <!DOCTYPE html>
                    <html>
                    <head>
                            <title>My First Webiste!</title>
                            <link type="text/css" rel="stylesheet" href="[your main website folder name]/[your css file name].css">
                        </head>
                        <body></body>
                </html>

The <title> tag will set your website's tab to whatever you put inside the tag. The <link> tag we put helps the computer find the CSS through the file path and link it to our HTML file. This will be useful to help style our website.TO VIEW YOUR WEBSITE, you double-click the HTML file from your main folder and it will open the file to your default browser. The browser will not update real-time so you will have to save your code first and then refresh the page.

Adding tags to the Body Tag

Right now the website seems empty. We should add some content. Let's add a heading and a paragraph to our website:

                <!DOCTYPE html>
                    <html>
                        <head>
                            <title>My First Webiste!</title>
                            <link type="text/css" rel="stylesheet" href="[your main website folder name]/[your css file name].css">
                        </head>
                        <body>
                            <h1>My first heading!>
                            <p>My first paragraph!>
                        </body>
                </html>

As you can see, the <h1> tag is the heading tag. There are many heading tags ranging from 1 to 6. 1 being the biggest font size and 6 being the smallest. If you haven't guessed yet the <p> is the paragraph tag. Another thing you can try is using the BOLD and EMPHASIS tags on a word or sentence by using <b>Bold text</b> and <em>Text in italics</em>. Try it.

Your first CSS script

Your website right now looks bland. We can change that with the power of CSS. Lets change the background colour and font for our website. Make sure you are typing out the code in your CSS file not in your HTML file!

            body {
                background: #73d625;
                font-family: Helvetica;
            }

As you can see, CSS has a way different syntax as HTML. The CSS code above tells the computer that EVERYTHING in the <body> tag that the whole website background will have a shade of green (background: #73d625;, you can use a colour picker to visualise hex or rgb colours) and all the text in it will have a "Helvetica" font which is same font used in this website.

Id and Class

Ids and Classes are very useful in HTML and CSS. Ids are used to define one specific element while Classes define multiple elements. Here is an example:


HTML:

            <!DOCTYPE html>
            <html>
                <head>
                    <title>My First Webiste!</title>
                    <link type="text/css" rel="stylesheet" href="[your main website folder name]/[your css file name].css">
                </head>
                <body>
                    <h1 class="headingClass">My first heading!>
                    <h1 class="headingClass">Another heading.>
                    <h2 class="headingClass">A smaller heading...>
                    <p id="paragraphID">My first paragraph!>
                </body>
            </html>

CSS:

            body {
                background: #73d625;
                font-family: Helvetica;
            }
            .headingClass {
                background: black;
                color: white;
            }
            #paragraphID {
                color: red;
            }

In the HTML code, all the heading tags have a class selector called "headingClass" and the one paragraph tag has an Id selector called "paragraphID". In the CSS code, all class selectors start with a ".", while the Id selectors start with a "#". As we can see in the CSS code, all the tags with the headingClass class will have a black background and its text will have a white colour. Meanwhile, the paragraph tag with a paragraphID will have its text in red.
